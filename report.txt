Отчет за неделю:
1) Программа выполняет основной функционал: авторизация, получение списка файлов, загрузка файла на сервер, загрузка
файла на клиент, удаление файла с сервера
2) Требуется:
 - провести рефакторинг - перенести основной код обработки команд из Хендлеров в новые классы FileTransfer (работа с
 файлами) и AuthTransfer (работа с авторизацией)
 - доработать клиентскую часть - добавить проводник клиентских файлов, возможность перелогиниться, сохранение настроек
 на клиенте, прогрессбар при загрузке/выгрузке файлов
 - доработать серверную часть - доработать авторизацию с использованием БД, улучшить обработку исключений, добавить
 логгер, добавить возможность менять положение родительской папки, расширить набор команд
 3) На клиетской стороне: переработано получение списка файлов, удаление файла, обработка успешного удаления файлов
